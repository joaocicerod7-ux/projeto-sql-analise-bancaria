**Projeto SQL – Análise de Dados Bancário**

**Sobre o Projeto**
Este projeto simula um sistema bancário com o objetivo de aplicar os meus conhecimentos em SQL.

**Conceitos SQL Aplicados**

SELECT
WHERE
ORDER BY
DISTINCT
INNER JOIN
LEFT JOIN
GROUP BY
HAVING
Funções agregadas (SUM, COUNT, AVG, MAX)
CASE
Subqueries

**Análises Realizadas**

1. Clientes com maior movimentação

Cálculo do total movimentado por cliente utilizando SUM e GROUP BY.

2. Classificação por perfil financeiro

Uso de CASE para classificar clientes como:

Alta Renda

Média Renda

Baixa Renda

3. Clientes acima da média

Utilização de subqueries para identificar:

Clientes com saldo acima da média geral

Clientes com movimentação acima da média

4. Identificação do maior valor de transação

Uso da função MAX em subquery.

**Objetivo do Projeto**

Consolidar conhecimentos em SQL intermediário aplicando:

Modelagem relacional

Análise de dados
